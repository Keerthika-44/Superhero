# 🦸 Superhero Landing Page

A modern and responsive superhero-themed landing page built using HTML, CSS, and JavaScript. The project features a fixed superhero background image that creates a smooth scrolling (parallax-like) effect.

## 🚀 Features

- Responsive design
- Fixed superhero background image
- Smooth scrolling effect
- Clean and modern UI
- HTML, CSS, and JavaScript only
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
superhero-landing-page/
│── index.html
│── style.css
│── script.js
│── images/
│    └── superhero.jpg
└── README.md
```
